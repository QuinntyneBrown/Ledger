Milestone badge

Ledger’s reward tile. A gemstone medal celebrates an achievement — a first weigh-in, a streak, a goal reached — and sits alongside the ones still locked to hint at what’s next. It’s the moment the numbers turn into encouragement.

50% of the way to goal

When to use it

Use a milestone badge to mark a meaningful, earned moment — not routine activity. Badges reward the habits Ledger is trying to build: showing up, trending down, reaching the goal. Grant them rarely enough that each one still feels good.

  • Show earned badges at full saturation; keep locked ones dimmed but visible so people can see what’s ahead.
  • Reserve the three medal tints for tiers or categories — gold for goals, teal for consistency, violet for progress.
  • Pair the award moment with a snackbar or celebratory dialog; the tile is where it lives afterward.

Accessibility. The medal icon is decorative, so the name and description must fully convey the achievement and its state — add “Locked” to the description (or an aria-label) rather than relying on the dimmed look. Don’t signal earned-versus-locked with color alone; the lock icon and copy carry it too.
