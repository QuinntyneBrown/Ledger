Typography

A minor-third scale on system-font stacks, with a step-out at the top for the hero weight number. The quiet star is tabular mono: every digit the same width, so columns of numbers and dates align perfectly. That is the "ledger" in Ledger.

The type scale

Eleven steps on a 1.200 ratio (minor third) that break into a larger stride at the display end. Each token is a rem so it respects the reader's root size.

Font roles

Three system stacks, zero network requests. Display and sans overlap on most platforms but split intent; mono is where the identity lives.

Display
Ag

Titles, hero weights, gauge values. Heavy and tight. --font-display — SF Pro Display, Segoe UI Variable.

Sans
Ag

Body, labels, controls. Neutral and legible. --font-sans — system-ui, Segoe UI, Roboto.

Mono
09

All numbers, dates, deltas, code. Tabular. --font-mono — SF Mono, Cascadia, JetBrains Mono.

The tabular numeral

Ledger's hero moment: the current weight, set in the display face at --text-6xl with tabular figures. Because every digit is one width, the number never shifts as it counts or updates.

type/hero-numeral
74.9 kg

The .au-numeral utility and .au-mono both set font-variant-numeric: tabular-nums. Apply one wherever numbers stack — a table column, a delta, a running total — so digits align on their decimal.

Weights

Five steps. Regular carries body; semibold does most of the emphasis; heavy is reserved for display numerals.

Aurora 400--weight-regular400 · body
Aurora 500--weight-medium500 · labels
Aurora 600--weight-semibold600 · emphasis
Aurora 700--weight-bold700 · headings
Aurora 800--weight-heavy800 · display

Line-height & letter-spacing

Tighter tracking as type gets bigger; more leading as lines get longer.

Utilities bundle the common recipes: .au-display, .au-h1/2/3, .au-lead, .au-caption, .au-eyebrow and .au-mono. Prefer them over hand-setting size + weight + tracking each time.
