Aurora Components Button
Actions & inputs

Button

Buttons commit an action. Aurora leads with a single luminous gold primary per view and steps down through tonal, outlined and text weights so the interface always shows one obvious next move.

When to use it

Use a button to perform an action — save an entry, confirm a goal, open a sheet. Choose the weight by importance: exactly one primary per view for the single most likely action, tonal or outlined for secondary paths, and text for low-stakes or dismissive actions. For anything destructive, reach for the danger variants.

  • Label with a verb — “Log weigh-in”, “Set goal”, “Delete entry” — never a bare noun.
  • Reserve the gold primary for the one action you want people to take.
  • Use an icon button for compact, recognizable actions in dense bars; give it an aria-label.
  • The FAB is the app’s persistent “add a weigh-in” affordance — one per screen.

Anatomy

1 2 3
  • 1 Container — a full pill (--radius-pill) with a 44px minimum target and a soft brand glow.
  • 2 Leading icon — optional, 18px, tuned to the label; sits before the text.
  • 3 Label — a short verb phrase in the semibold sans face.

Do & don’t

Do
One primary action, paired with a quiet text button for the way out.
Don’t
Don’t stack several primaries — competing gold buttons erase the hierarchy.

Full-width (block)

On the narrow app frame, primary actions stretch to the full column with .au-btn--block.

button/block

States

Hover lifts the button one pixel and brightens the fill; active presses it back down. .is-loading swaps the label for a spinner and blocks input; :disabled dims it to 45%.

button/states
<button class="au-btn au-btn--primary is-loading">Saving</button>
<button class="au-btn au-btn--primary" disabled>Disabled</button>

Icon buttons

Icon-only actions for toolbars and headers. Always supply an aria-label. Available in default (44px) and small (36px).

button/icon
<button class="au-icon-btn" aria-label="Edit entry">
  <svg class="au-icon"><use href="#i-edit"/></svg>
</button>
<button class="au-icon-btn au-icon-btn--sm" aria-label="Delete entry">
  <svg class="au-icon"><use href="#i-trash"/></svg>
</button>

Floating action button

The signature “add” affordance. Use the extended FAB with a label for the primary flow, or the round .au-fab--round where space is tight.

button/fab
<button class="au-fab">
  <svg class="au-icon"><use href="#i-plus"/></svg> Weigh in
</button>
<button class="au-fab au-fab--round" aria-label="Add weigh-in">
  <svg class="au-icon"><use href="#i-plus"/></svg>
</button>
