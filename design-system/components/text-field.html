Actions & inputs

Text field

A text field captures a single value. Aurora pairs a calm label with a generous 52px input, a clear focus ring, and validation that speaks in colour and words — never colour alone.

We’ll only use this to sync your entries.

When to use it

Use a text field for free-form or short structured input — an email, a name, a note. For a single number with a target (a weight), prefer the big-number entry; for a choice from a fixed set, use a Select.

  • Always give a persistent label above the field — placeholders are not labels.
  • Mark required fields with the .au-req asterisk and say why in helper text.
  • On error, set .is-error and explain the fix in an .au-help.is-error line.
  • Use a leading icon or unit affix to pre-frame the expected value.

Anatomy

1 2 3 4
  • 1 Label — a persistent field name with an optional required marker.
  • 2 Leading icon — an optional glyph that frames the value type.
  • 3 Affix — a trailing unit or symbol in the mono face.
  • 4 Helper — guidance or a validation message beneath the input.

Do & don’t

Do
Keep the label visible and use the placeholder only for an example.
Don’t
Don’t rely on a placeholder as the label — it vanishes the moment someone types.

Default, filled & focus

The resting field hovers to a stronger border; focus draws a 4px gold ring via :focus-within. The third field below simulates that focus state.

field/base 
<div class="au-field">
  <label class="au-label" for="name">Filled</label>
  <div class="au-input-wrap">
    <input class="au-input" id="name" value="Alex Rivera">
  </div>
</div>

Error & success

.is-error tints the field coral and pairs with an .au-help.is-error message; .is-success confirms with a teal border.

field/validation

Looks good — address verified.
<div class="au-field is-error">
  <label class="au-label" for="email">Email</label>
  <div class="au-input-wrap"><input class="au-input" id="email"></div>
  <div class="au-help is-error">
    <svg class="au-icon"><use href="#i-alert-circle"/></svg>
    Enter a valid email address.
  </div>
</div>

Big-number entry

The signature weigh-in control: an oversized display numeral with a mono unit. Built for a single, confident number.

field/bignum
kg 
<div class="au-bignum-field">
  <input class="au-bignum au-numeral" value="82.4"
         inputmode="decimal" aria-label="Weight">
  <span class="au-bignum-unit">kg</span>
</div>
