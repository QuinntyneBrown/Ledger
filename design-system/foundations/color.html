Foundations

Color

A cool near-black spine of ink, a single luminous gold accent, and a small set of status hues. Colour in Aurora carries meaning — teal is progress, coral is danger — and the signature aurora gradient ties it all together.

Ink — the spine

A 14-step cool near-black to near-white ramp. Never pure #000; the slight blue keeps large dark fields from feeling dead. Ink defines every surface, border and text colour through the semantic layer.

Semantic tokens

The layer components actually read. Each maps to a primitive per theme — flip the theme and the meaning holds while the value changes. Toggle the theme (top right) to watch these previews follow.

TokenVariableRole
--canvasThe page backdrop — deep ink in dark, off-white in light.
--surface-1Default card / panel surface, one step above the canvas.
--surface-2Raised controls: inputs, chips, hover fills.
--surface-3Tracks and the highest inset surface.
--text-hiPrimary text — headings, values, anything that must read first.
--text-midBody copy and secondary labels.
--text-loCaptions, meta, placeholders, disabled affordances.
--brandThe gold accent: primary actions, focus, active nav.
--goodPositive / on-track — a loss toward goal, success.
--warnWarning — a plateau, an approaching limit.
--badDanger / destructive / error — off-track, delete.
--infoNeutral information — tips, non-urgent notices.
--borderHairline separators and card edges (a low-alpha white/ink).

Each status hue also ships a --*-subtle tint (for fills) and a --*-text variant tuned for legible text on the canvas. Reach for --good-text on a label, --good-subtle behind it.

The aurora gradient

The system's namesake: a violet → teal → gold sweep. It fills the signature dial, brand marks and progress bars — the colour itself communicates a journey from just-started to nearly-there.

Two themes, one meaning

Ledger is dark-first; the docs default to what you are reading now. Both are the same semantic tokens with a different primitive mapping. Nothing in a component changes.

Light −0.4 kg Save
Contrast & AA. Text tokens are tuned to clear WCAG AA (4.5:1 for body, 3:1 for large text) on their intended surface in both themes. The gold accent uses dark text on top (--text-on-gold) rather than white, which would fail. Never signal with colour alone — pair every status hue with an icon or label. See Accessibility.
