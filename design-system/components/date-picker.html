Date picker

Choose a day. A calendar-icon field opens an inline month grid where the selected day fills gold, today wears a brand ring, and days from neighbouring months fade back.

When to use it

Use a date picker when the value is a calendar day the person recognizes on sight — a goal deadline, a start date, the day of a weigh-in. The field alone is enough for a keyboard-first flow; the inline grid helps when someone is choosing relative to “today” or a weekend. For a pure numeric value, use a field; for one of a few preset ranges, a segmented control.

  • Show the chosen date back in a human, tabular format — “24 Jul 2026”.
  • Mark today with a ring and the selection with the gold fill — never the same treatment.
  • Fade out-of-month days so the current month stays the figure.
  • Disable impossible days (before today for a target) rather than letting them be picked.

Do & don’t

Do
Echo the picked date in a clear, tabular format people can read at a glance.
Don’t
Don’t surface a raw machine timestamp — format it for a human.

Date field

A read-only field with a leading calendar glyph and a trailing chevron that opens the grid. The middle field simulates focus.

date/field 
<div class="au-input-wrap">
  <svg class="au-icon"><use href="#i-calendar"/></svg>
  <input class="au-input au-numeral" value="24 Jul 2026" readonly>
  <svg class="au-icon"><use href="#i-chevron-down"/></svg>
</div>

Structure & classes

The date field reuses the standard .au-input-wrap. The inline calendar below is a documented pattern built from tokens — drop these class names into your own grid.

ClassElementRole
.au-input-wrap + #i-calendardivThe trigger field; a leading calendar icon and a formatted value.
.caldivCalendar container (pattern class).
.cal-head / .cal-titlediv / spanMonth label between stepper icon buttons.
.cal-griddiv7-column CSS grid holding the weekday row and day cells.
.cal-dowdivA weekday header cell in the mono face.
.cal-daybuttonA single day; carries the state classes below.

Day states

ClassMeaningTreatment
.cal-dayIn-month dayMid-tone numeral; hover raises a surface.

Accessibility. Give the grid role="grid" and an accessible month label, mark the current day with aria-current="date" and the chosen one with aria-selected="true". Because today and the selection use different visuals, they never collide. Keep a plain typed-date field available for keyboard-only entry.
