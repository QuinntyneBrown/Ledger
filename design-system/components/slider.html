Slider

A slider sets a value along a continuous range. The track fills gold up to the thumb so the current position reads at a glance — ideal for a goal pace or a target you tune by feel.

Goal pace 0.5 kg / wk
GentleAggressive

When to use it

Reach for a slider when the exact number matters less than the feel of a value along a range — a goal pace, a target band, a reminder time. When precision is essential, pair the slider with a numeric field that mirrors it. When there’s a hard target and the story is “how close am I?”, the value belongs in a Gauge, not a slider.

  • Always show the current value in a live read-out beside the track.
  • Label the two ends so the direction of travel is unmistakable.
  • Set a sensible step — round numbers beat pixel-perfect precision.
  • The fill is driven by a --_pct custom property; keep it in sync with the value.

Anatomy

1 2 3
  • 1 Fill — the gold portion from the start to the thumb.
  • 2 Track — the remaining range in a muted surface.
  • 3 Thumb — a 24px grabbable handle ringed against the canvas.

The rotary cousin — a dial

When a value curls back on itself or wants to feel like a physical knob, use the radial Gauge as a dial input. Same aurora fill, wrapped into a circle. The linear slider and the rotary dial are two readings of one idea.

Values across the range

The fill grows with the value. Set --_pct to match the thumb.

slider/values 
<div class="au-slider">
  <input type="range" min="0" max="100" value="70"
         style="--_pct:70%" aria-label="Target">
</div>

Accessibility. The native range input is keyboard-operable with the arrow keys out of the box. Give it an aria-label or a bound <label>, and expose the current value in visible text so it isn’t carried by the thumb position alone. For a circular dial input, mirror this with role="slider" and aria-valuenow on the Gauge.
