Aurora Components Switch
Actions & inputs

Switch

A switch flips a single setting on or off and applies it immediately — no save button. The track fills gold when on and the thumb glides across on a springy ease.

When to use it

Use a switch for a binary setting that takes effect the instant it is toggled — notifications, dark mode, weekly summaries. If the change should only apply after an explicit save, or the choice is one of several, use a checkbox or radio instead.

  • Toggling is immediate — never require a separate confirm step.
  • Pair each switch with a clear label, and a description when the effect isn’t obvious.
  • Write the label as the on state — “Daily reminder”, not “Reminders on/off”.
  • For a yes/no answer inside a form that is saved later, use a checkbox.

Anatomy

1 2
  • 1 Track — the pill; neutral when off, gold when on.
  • 2 Thumb — slides 20px on toggle and shifts colour to sit on the fill.

Do & don’t

Do
Daily reminder
Label the setting by its on state and let the toggle apply at once.
Don’t
Don’t make a switch wait on a separate Apply button — that’s a checkbox.
