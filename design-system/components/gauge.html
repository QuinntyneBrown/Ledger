Aurora Components Gauge
Data display · Signature

Gauge

A radial dial that shows how close a value is to its goal. Its arc is filled with the aurora gradient — violet at the start, gold at the finish — so the colour itself communicates progress before a single number is read.

0%
to goal

When to use it

Reach for a Gauge when a single value has a target and the story is “how far along am I?”. In Ledger it is the hero of the dashboard: percent of the way from the starting weight to the goal weight. Use exactly one gauge per view — it is a focal point, not a repeatable stat.

  • Use the full ring (360°) for progress toward a goal.
  • Use the 270° gauge when the value is a rate or score without a hard maximum.
  • Pair the arc with a large tabular numeral in the center — never rely on colour alone.
