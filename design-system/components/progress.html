Progress

Four ways to say “working on it.” Use a determinate bar when you know how far along you are, a spinner when you don’t, and skeletons to hold a layout while its data loads.

To goal weight 72%
75%

When to use each

  • Bar (.au-progress) — a determinate task with a known percentage: import, upload, “X of Y logged.” The fill is the aurora gradient by default.
  • Spinner (.au-spinner) — a short, indeterminate wait where a percentage is unknown. Use --lg for a full-view load.
  • Skeleton (.au-skeleton) — a shimmering placeholder that mirrors the shape of the content that is loading, so the layout doesn’t jump.
  • Ring (.au-ring) — a tiny CSS-only radial for an inline sub-goal or streak. For the hero dial, use the Gauge.

Anatomy of the bar

1 2
  • 1 Fill — the aurora gradient, width ∝ value, animated on change.
  • 2 Track — the remaining path in a muted --surface-3.

Do & don’t

Do
Use a bar when the percentage is known.
Don’t
Don’t fake a bar for an unknown wait — use a spinner.

Determinate bars

The default fill is the aurora gradient; add .au-progress--brand for a solid gold bar.

progress/bars 
<div class="au-progress">
  <div class="au-progress-bar" style="width:60%"></div>
</div>

<!-- solid gold -->
<div class="au-progress au-progress--brand">
  <div class="au-progress-bar" style="width:88%"></div>
</div>

Spinners

progress/spinner
<span class="au-spinner"></span>
<span class="au-spinner au-spinner--lg"></span>

Skeleton loading card

Mirror the final layout with --circle and --text placeholders so nothing shifts when data arrives.

progress/skeleton
<div class="au-card">
  <span class="au-skeleton au-skeleton--circle"
        style="width:44px;height:44px"></span>
  <span class="au-skeleton au-skeleton--text" style="width:55%"></span>
  <span class="au-skeleton" style="height:72px"></span>
</div>

Compact ring

A CSS-only conic ring for inline sub-goals. Set the percentage on --_pct.

progress/ring
33%
66%
100%
<div class="au-ring" style="--_pct:66"><span>66%</span></div>

Tokens consumed

TokenApplied to
--auroraDefault bar fill.
--brandSolid bar, spinner accent, ring fill.
--surface-3Bar track, spinner base, ring remainder.
--borderSkeleton shimmer sweep.
--dur-slow · --ease-standardBar width transition.
Accessibility. Give a determinate bar role="progressbar" with aria-valuenow/min/max; give a spinner role="status" and an aria-label like “Loading”. Skeletons are decorative — mark their container aria-hidden="true" and expose a single polite “loading” message instead. All animation collapses under prefers-reduced-motion.
