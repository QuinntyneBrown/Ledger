Empty state

What a view shows when it has nothing to show. A good empty state is an invitation, not a dead end — an illustrated tile, a short title, one line of guidance, and a single clear next action.

No weigh-ins yet

Log your first weight to start your ledger and watch the dial come to life.

When to use it

Show an empty state whenever a list, chart, or search would otherwise render a blank space. There are three distinct moments, and the tone differs for each.

  • First run — there is no data yet. Be encouraging and offer the primary action to create the first record.
  • No results — a filter or search matched nothing. Be neutral and offer a way to clear or broaden the query.
  • All done — the user cleared everything. Be affirming; a completed list is a small win.
  • Keep it to one action. Two competing buttons defeat the purpose of a focused prompt.
